ASGL page at UCSF

ASGL is a program for preparing all sorts of PostScript plots from simple data files. The current version is 1.3.2 and was released on November 9th, 2006.

README file contains more information about ASGL.

The source code for ASGL is available on GitHub. For users of Fedora Core 6 (or similar) Linux systems, source and binary RPMS are also available. (These packages are GnuPG-signed for security, using this key.)
